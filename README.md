## Overview / نظرة عامة
A concise, action-first roadmap to get from zero to competent in cyber security.  
خارطة طريق عملية ومباشرة تساعدك تبدأ وتتمرن وتتخصص.

---

## Quick Roadmap (Steps) / الخريطة السريعة
1. Programming / البرمجة  
2. Networking & Protocols / الشبكات والبروتوكولات  
3. Operating Systems (Linux & Windows) / أنظمة التشغيل  
4. Tools & Platforms / أدوات ومنصات  
5. Fundamentals: Web, Crypto, Forensics / أساسيات: الويب، التشفير، الأدلة الجنائية  
6. Choose a Path / اختر المسار  
   - Red Team (Offensive) / الفريق الأحمر  
   - Blue Team (Defensive) / الفريق الأزرق  
   - GRC (Governance, Risk & Compliance) / الحوكمة وإدارة المخاطر  
7. Practical Practice / التطبيق العملي (CTFs, labs, writeups)  
8. Portfolio & GitHub / البورتفوليو وجيتهاب  
9. Apply & Improve / قدم وطور مهاراتك باستمرار

---

## 30 / 60 / 90 Plan / خطة 30/60/90 يوم
**Days 1–30 (Foundations / أساسيات):**
- 1. Learn Python basics (variables, control flow, functions).  
- 2. Basic Linux commands and file system.  
- 3. Networking fundamentals: IP, TCP/UDP, common ports.  

**Days 31–60 (Core Skills / مهارات أساسية):**
- 4. Web basics: HTTP, HTML, simple JS.  
- 5. Practice with TryHackMe beginner paths and OverTheWire Bandit.  
- 6. Start using Nmap, Wireshark, Burp Suite (community).  

**Days 61–90 (Specialize & Apply / تخصّص وتطبيق):**
- 7. Choose path (Red / Blue / GRC).  
- 8. Join CTFs, solve challenges, write writeups.  
- 9. Publish 2–3 small projects on GitHub.

## Study Checklist (per step) / قائمة مراجعة لكل خطوة
**1. Programming**
- Resources: Python tutorial, practice exercises.
- Deliverable: 3 small scripts on GitHub.

**2. Networking**
- Topics: IP/subnet, DNS, TCP/UDP.
- Deliverable: Documented Nmap scans + notes.

**3. Operating Systems**
- Linux: shell, permissions, users.
- Windows: PowerShell basics, Event Viewer.
- Deliverable: Lab notes + commands cheat-sheet.

**4. Tools & Platforms**
- Install and test: Nmap, Wireshark, Burp Suite, Git.
- Platforms: TryHackMe (roadmap), HackTheBox, picoCTF.
- Deliverable: Completed beginner rooms and screenshots.

**5. Fundamentals**
- Web: XSS, SQLi, auth flaws.
- Crypto: basic enc/dec, hashing.
- Forensics: basic file/memory artefacts.
- Deliverable: At least one writeup per category.

**6. Choose Path**
- Red: web + pwn + exploitation labs.
- Blue: logging + SIEM + detection rules.
- GRC: frameworks + risk assessments.
- Deliverable: Path-specific mini-project (tool, detection, or policy).

## Practice & CTF Workflow / التدريب واستراتيجية الـCTF
1. Pre-CTF: assign roles (web, pwn, crypto, recon).  
2. During: document commands, capture flags, communicate updates.  
3. Post-CTF: write a full writeup, record lessons learned.  
4. Keep a repo `writeups/` with date and category tags.
